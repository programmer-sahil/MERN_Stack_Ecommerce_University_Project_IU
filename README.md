# E-Commerce Website (MERN Stack)

## Project Overview

This project is an e-commerce platform built using the MERN stack (MongoDB, Express.js, React, and Node.js). It provides users with a seamless experience to shop for products, create accounts, manage orders, and make secure payments using PayPal. Store administrators can manage products, customers, and orders efficiently.

## Tech Stack

**Frontend:**
- React.js
- Vite (for optimized build speeds)
- TailwindCSS (for responsive design)
- Axios (for API requests)
- React-Router-Dom (for page routing)

**Backend:**
- Node.js
- Express.js
- MongoDB (via Mongoose for data modeling)
- Bcryptjs (for password encryption)
- JSON Web Token (for user authentication)

**External Services:**
- PayPal SDK (for secure payments)
- Cloudinary (for image storage and management)
